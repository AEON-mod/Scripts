PC Font Changer ⚡
Quickly change your Windows PC fonts using simple registry tweak files! No coding needed.
🌟 Features
Feature	Description

🎨 Custom Fonts	Change to any installed font on your PC

🚀 One-Click	Just rename .txt to .reg and run

↩️ Easy Revert	Restore default fonts instantly

🪟 Windows 10/11	Tested and working on latest Windows📝 Simple Edit	Only need to type your font name

📋 How to Use
Step 1: Install Your Font First
Download font file (.ttf or .otf)

Right-click → Install (or double-click → Install button)

Verify font appears in Control Panel > Fonts

Step 2: Change to Custom Font
Download font-changer.txt

Open with Notepad

Find this line:

text
YOUR_FONT_NAME_HERE
Replace with EXACT font name (case-sensitive!)
✅ Correct: Segoe UI, Arial, Roboto

❌ Wrong: segoe ui, ARIAL, roboto light

Save and rename file from font-changer.txt → font-changer.reg

Double-click .reg file → Click Yes → Yes

Restart your PC (critical!)

Step 3: Revert to Default Fonts
Download font-changer-to-default.txt

Do NOT edit this file

Rename to font-changer-to-default.reg

Double-click → Yes → Yes

Restart PC
⚠️ Critical Warnings
Do ✅	Don't ❌
Edit font-changer.txt	Edit font-changer-to-default.txt
Use exact font name spelling	Change capitalization
Create restore point first	Skip the restart
Run as Administrator if blocked	Force restart mid-process
How to Find Exact Font Name
Press Win + R → type control fonts → Enter

Find your font → copy the exact name

Paste into font-changer.txt

🔧 Troubleshooting
Problem	Solution
Font name not working	Check exact spelling in Fonts control panel
Registry edit blocked	Right-click .reg → Run as Administrator
No visual changes	Restart explorer.exe or full reboot
System unstable	Run font-changer-to-default.reg immediately
Permission denied	Create system restore point first
Quick Fix Script
If changes don't apply, open Command Prompt as Admin and run:

text
taskkill /f /im explorer.exe
start explorer.exe

Want to use "Roboto" font?

text

YOUR_FONT_NAME_HERE

Roboto

text
Segoe UI SemiLight

⚡ Quick Command Reference
bash
# Windows PowerShell to find installed fonts
Get-ItemProperty -Path "HKLM:\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Fonts" | 
  Select-Object -ExpandProperty Property | 
  Sort-Object
🛡️ Safety First
Create Restore Point:

Press Win + R → sysdm.cpl → System Protection → Create

Backup Registry:

Press Win + R → regedit → File → Export → Save

Test on Virtual Machine first (recommended)
